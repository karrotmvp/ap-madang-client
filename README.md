# 랜선 동네 모임 FE

> MVP Team6 온라인 모임팀 Front-end 개발 Dave


## 시작하기


### 환경 세팅
- `.env`를 채워주세요

``` javascript
// /.env
    APP_ID=''
    API_URL=''
    MINI_PRESET_URL=''
    GA_KEY=''

    FA_KEY=""
    FA_AUTH_DOMAIN=""
    FA_PROJECT_ID=""
    FA_STORAGE_BUCKET=""
    FA_MESSAGING_SENDER_ID=""
    FA_APP_ID=""
    FA_MEASUREMENT_ID=""

    AGORA_APP_ID=""
    AGORA_TOKEN= ""

    INFO_NOTION_URL=""

    KARROT_SCHEME=""
 ```


### 실행

``` javascript
npm i

//dev
npm run dev

// production
npm run build
 ```
